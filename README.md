# Coach OS

نظام إدارة عملاء مدرب جيم شخصي — تطبيق PWA يعمل على الموبايل والتابلت، بدون إنترنت (offline-first)، مع مزامنة سحابية عبر Google Sheets.

## المميزات
- إدارة المتدربين، الاشتراكات، الجلسات، والمدفوعات
- تتبع القياسات والتقدم مع رسوم بيانية
- موسوعة عضلات وتمارين تفاعلية (أطلس جسم SVG)
- خريطة تعافي لكل متدرب حسب آخر تدريب
- يعمل بالكامل بدون نت، ويتزامن تلقائيًا مع Google Sheets عند توفر الاتصال

## الإعداد
راجع [DEPLOY.md](DEPLOY.md) لخطوات ربط الباك اند (Google Apps Script + Google Sheet).

## التقنية
ملف HTML واحد بدون أي build step — Vanilla JS، Chart.js للرسوم البيانية، وGoogle Apps Script كباك اند.
